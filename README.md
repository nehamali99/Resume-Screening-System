# Resume Screening System

## 📌 Project Overview

The Resume Screening System is a Java-based application designed to automate the resume evaluation process. The system analyzes candidate resumes, extracts relevant information, and ranks applicants based on predefined job requirements. This helps recruiters and HR professionals save time and improve hiring efficiency.

## 🚀 Features

* Resume Upload (PDF Format)
* Candidate Information Extraction
* Keyword-Based Resume Analysis
* Resume Scoring System
* Candidate Ranking
* User-Friendly Graphical Interface
* PDF Report Generation
* Fast and Accurate Screening

## 🛠️ Technologies Used

* Java
* Java Swing (GUI)
* Apache PDFBox
* File Handling
* Object-Oriented Programming

## ⚙️ How to Run

1. Install Java JDK 8 or later.
2. Open the project in Eclipse IDE.
3. Add required libraries (PDFBox).
4. Compile and run the main Java file.
5. Upload candidate resumes.
6. View scores and rankings.
